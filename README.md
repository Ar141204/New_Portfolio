# Abdul Rahman M | Portfolio

Welcome to the source code for my personal portfolio! This is a fast, responsive, and visually distinct single-page portfolio built to showcase my skills, projects, and professional journey as a Full Stack Developer.

## 🎨 Theme & Aesthetics
The portfolio utilizes a highly customized **"Retro Amber Accents"** light theme. It borrows heavily from pixel-art logic and brutalist UI elements, featuring:
- Sharp box-shadows (`Xpx Xpx 0px`) and pixelated border accents.
- A "Player Profile" / "Quest Log" RPG metaphor for standard sections.
- Micro-animations and custom SVG assets mimicking classic computer interfaces (like the Polaroid profile picture).
- 100% Vanilla HTML, CSS, and JavaScript with no heavy external UI libraries.

## 🛠️ Built With
- **HTML5:** Semantic structure and accessibility.
- **Vanilla CSS3:** Native CSS variables for easy theming, custom keyframe animations, and Flexbox/Grid layouts.
- **Vanilla JavaScript:** For smooth scrolling, scroll-based animations (IntersectionObserver), and form handling.

## 🚀 Key Sections
1. **Hero Section:** Features a dynamic introduction with social links, a resume download button, and a customized Polaroid profile image.
2. **Player Stats / Status:** Quick overview of my current role, location, and professional availability.
3. **About Me:** Highlights core competencies including FullStack development, Automation, AI Infrastructure, and Collaboration.
4. **Quest Log (Experience):** A timeline of my professional experience, including my current role as AI Division Lead at Kutty Pixel Inc.
5. **Training Grounds (Education):** My academic background in Computer Science at Hindustan University.
6. **Featured Builds (Projects):** Details on significant projects like the Water Quality Forecast System, Traffic Density Analyzer, and Car Price Predictor.
7. **Power Ups (Skills):** A breakdown of my languages, frameworks, tools, and soft skills.
8. **Achievements Unlocked (Certifications):** Badges and certifications earned from AICTE, GUVI, HackerRank, etc.
9. **Contact:** An integrated contact form that gracefully opens the user's default email client.

## 💻 Running the Project
Since this is a vanilla web project, running it locally is incredibly simple:

1. Clone the repository to your local machine.
2. Open `Rahman.html` in your favorite web browser. 

Alternatively, if you have Python installed, you can serve it via a local development server:
```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000/Rahman.html`.

## 🔒 Image Protection
The profile image is secured via CSS and HTML attributes (`pointer-events: none`, `user-select: none`, `draggable="false"`, `oncontextmenu="return false;"`) to deter casual copying or downloading.
